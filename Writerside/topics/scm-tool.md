# 형상관리 툴

소프트웨어 버전을 관리하기 위한 툴이다.  
형상관리는 소스의 변화를 지속적으로 관리하고, 소스는 버전별로 관리가 된다.  
다수의 사용자가 동일한 소스를 문제없이 사용할 수 있도록 만들어주며,  
버전에 문제 발생 시, 되돌아 갈 수 있는 방법도 제시한다.

## 종류
| Client/Server Type | Distribute Repository Type |
|--------------------|----------------------------|
| CVS, SVN, TFS      | Git                        |
